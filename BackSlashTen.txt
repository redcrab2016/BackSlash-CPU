BackSlash 10
8/16 bits data. 8 bits is a byte, 16 bits is a word.
16 bits address.
LSB means Less Significant Byte value of a word.
MSB means Most Significant Byte value of a word.
Each memory address represent a byte. 
16 bits data (not instruction) value is stored in memory little-endian: LSB is stored in an address before the address of the MSB.
Each instruction is represented by a word, stored in memory big-endian:  MSB value is stored in an address before the address of the LSB value. MSB value address is even, LSB value address is odd.

1) CPU Registers:
16 16-bits registers, from W0 to W15.
16 8-bits registers, from B0 to B15
The 8-bits registers from B0 to B15 are embedded inside the registers W0 to W7, and can be designated with an alias name.

B0 (alias L0) is LSB of W0
B1 (alias H0) is MSB of W0
B2 (alias L1) is LSB of W1
B3 (alias H1) is MSB of W1
B4 (alias L2) is LSB of W2
B5 (alias H2) is MSB of W2
B6 (alias L3) is LSB of W3
B7 (alias H3) is MSB of W3
B8 (alias L4) is LSB of W4
B9 (alias H4) is MSB of W4
B10 (alias L5) is LSB of W5
B11 (alias H5) is MSB of W5
B12 (alias L6) is LSB of W6
B13 (alias H6) is MSB of W6
B14 (aliases L7 and ONE8) is LSB of W7
B15 (aliases H7 and ZERO8) is MSB of W7
Each 16 bits registers from W7 to W15 have one or more alias name.
ONE16 is alias of W7
ZERO16 is alias of W8
ST is alias of W9
SF is alias of W10
SA is alias of W11
DA is alias of W12
RA is alias of W13
FG is alias of W14
PC is alias of W15

The register W7 is always with value 1 (can't be altered), therefore the register aliases ONE8 ONE16 have always the value 1, alias ZERO8 have always have the value 0.
The register W8 is always with value 0 (can't be altered), therefore the register aliases ZERO16 have always the value 0.
The register W15 (alias PC) is dedicated to represent the address of the next instruction to execute. only even address. If it does represent an odd address, the less significant bit of register W15 is ignored (W15 valued as W15 - 1).
If the register W15 (alias PC) have value 65534 or 65535 (0xfffe or 0xffff in hexadecimal) the CPU halt.
The register W14 (alias FG) is dedicated to keep the flags.(TODO: explain each bit in flag register)
The register W13 (alias RA) is set with the value of register W15 (alias PC) before register W15 is modified by a LOAD or CPY operations.
The register W12 (alias DA) is conventionally used to represent a destination address
The register W11 (alias SA) is conventionally used to represent a source address
The register W10 (alias SF) is conventionally used to represent a secondary stack pointer address (often based/relative to main stack pointer) where temporary values (variable) are stored. (stack frame for local variables)
The register W9 (alias ST) is conventionally used to represent the main stack pointer address.
At reset time all register are set to value 0 except W7 (then B4 also) is always with the value 1.

2) Addressing mode:
Address used by LOAD and STORE instructions are represented between brackets [].
mode 0: [Wx] or [-Wx] or [+Wx] or [Wx-] or [Wx+] , represent the byte or word at address of the value stored in register Wx with x from 8 to 15.[-Wx] and [+Wx] indicates that Wx is decremented, incremented respectively before evaluation of Wx. [Wx-] and [Wx+] indicates that Wx is decremented, incremented respectively after evaluation of Wx. The increment and decrement quantity depends of the involved data size to transfer, 1 for 8-bits(byte) , 2 for 16-bits(word).
mode 1: [Wx + Wy] , represent the byte or word at address resulting in the sum of values stored in register Wx and Wy with x from 8 to 15 and y from 0 to 7
mode 2: [Wx + imm] : represent the byte or word at address resulting in the sum of values stored in register Wx and imm with x from 8 to 15. if imm = 0 then use mode 0
mode 3: [Wx + Wy + imm], represent the byte or word at address resulting in the sum of values stored in register Wx and Wy and imm with x from 8 to 15 and y from 0 to 7.if imm = 0 then use mode 1.
If the operation involve a 8-bits value transfer then imm is 1 or 2 or 3 or 4 or 5 or 6 or 7 or 8.
If the operation involve a 16-bits value transfer then imm is 2 or 4 or 6 or 8 or 10 or 12 or 14 or 16.


3) Instruction set
3.1) transfer value from register to register. No flag modified
	0000 000 r yyyy xxxx
		r : register size, value 0 for 8-bits register from B0 to B15, value 1 for 16-bits register from W0 to W15
		yyyy : source register
		xxxx : destination register
		Assembler template : 
			CPY Bx, By ; store register 'By' value to register 'Bx'
			CPY Wx, Wy ; store register 'Wy' value to register 'Wx'
		Assembler examples:
			CPY B3, B4  ; The value of register 'B4' is stored in register 'B3'  . Binary coded as '0000 000 0 0100 0011'
			CPY W5, W10 ; The value of register 'W10' is stored in register 'W5'  . Binary coded as '0000 000 1 1010 0101'
3.2) transfer value from memory to register (LOAD) , register to memory (STORE).
	Z,S flags are modified for LOAD operation.
	0001 d p ii yyy r xxxx : LOAD and STORE with addressing mode 0
		d: direction, 0 from memory to register (LOAD), 1 from register to memory (STORE)
		r: register size, value 0 for 8-bits register from B0 to B15, value 1 for 16-bits register from W0 to W15
		xxxx : register, if d=0  register is destination.If d=1 register is source, register B0 to B15 if r=0, register W0 to W15 if r=1
		yyy : register containing the address. the register is yyy+8 to designate W8 to W15.
		p : =0 increment W(yyy+8) after transfer, =1 increment W(yyy+8) before transfer.
		ii :  increment value of W(yyy+8) : 00 = no change, 01 : 1<<r increment, 11 : 1<<r decrement, 10: no change
		Assembler template :
			STORE [Wy], Bx ; store the 8-bits value of register Bx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15
			STORE [Wy], Wx ; store the 16-bits value of register Wx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15
			LOAD Bx, [Wy] ; store the 8-bits value stored in memory at address defined by value of register Wy to register Bx. x from 0 to 15 and y from 8 to 15
			LOAD Wx, [Wy] ; store the 16-bits value stored in memory at address defined by value of register Wy to register Wx. x from 0 to 15 and y from 8 to 15
			STORE [+Wy], Bx ; Wy = Wy + 1 then store the 8-bits value of register Bx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15
			STORE [+Wy], Wx ; Wy = Wy + 2 then store the 16-bits value of register Wx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15
			LOAD Bx, [+Wy] ; Wy = Wy + 1 then store the 8-bits value stored in memory at address defined by value of register Wy to register Bx. x from 0 to 15 and y from 8 to 15
			LOAD Wx, [+Wy] ; Wy = Wy + 2 then store the 16-bits value stored in memory at address defined by value of register Wy to register Wx. x from 0 to 15 and y from 8 to 15
			STORE [-Wy], Bx ; Wy = Wy - 1 then store the 8-bits value of register Bx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15
			STORE [-Wy], Wx ; Wy = Wy - 2 then store the 16-bits value of register Wx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15
			LOAD Bx, [-Wy] ; Wy = Wy - 1 then store the 8-bits value stored in memory at address defined by value of register Wy to register Bx. x from 0 to 15 and y from 8 to 15
			LOAD Wx, [-Wy] ; Wy = Wy - 2 then store the 16-bits value stored in memory at address defined by value of register Wy to register Wx. x from 0 to 15 and y from 8 to 15
			STORE [Wy+], Bx ; store the 8-bits value of register Bx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15, then Wy = Wy + 1
			STORE [Wy+], Wx ; store the 16-bits value of register Wx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15, then Wy = Wy + 2
			LOAD Bx, [Wy+] ; store the 8-bits value stored in memory at address defined by value of register Wy to register Bx. x from 0 to 15 and y from 8 to 15, then Wy = Wy + 1
			LOAD Wx, [Wy+] ; store the 16-bits value stored in memory at address defined by value of register Wy to register Wx. x from 0 to 15 and y from 8 to 15, then Wy = Wy + 2
			STORE [Wy-], Bx ; store the 8-bits value of register Bx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15, then Wy = Wy - 1
			STORE [Wy-], Wx ; store the 16-bits value of register Wx to memory address defined by value of register Wy. x from 0 to 15 and y from 8 to 15, then Wy = Wy - 2
			LOAD Bx, [Wy-] ; store the 8-bits value stored in memory at address defined by value of register Wy to register Bx. x from 0 to 15 and y from 8 to 15, then Wy = Wy - 1
			LOAD Wx, [Wy-] ; store the 16-bits value stored in memory at address defined by value of register Wy to register Wx. x from 0 to 15 and y from 8 to 15, then Wy = Wy - 2
		Assembler examples:
			STORE [W12], B5 ; the 8-bits value of register 'B5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 0 00 100 0 0101'
			STORE [W12], W5 ; the 16-bits value of register 'W5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 0 00 100 1 0101'
			LOAD B5, [W12] ; the 8-bits value at address defined by value of register W12 is stored in register B5. Binary coded as '0001 0 0 00 100 0 0101'
			LOAD W5, [W12] ; the 16-bits value at address defined by value of register W12 is stored in register W5. Binary coded as '0001 0 0 00 100 1 0101'
			STORE [+W12], B5 ; W12 = W12 + 1 then the 8-bits value of register 'B5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 1 01 100 0 0101'
			STORE [+W12], W5 ; W12 = W12 + 2 then the 16-bits value of register 'W5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 1 01 100 1 0101'
			LOAD B5, [+W12] ; W12 = W12 + 1 then the 8-bits value at address defined by value of register W12 is stored in register B5. Binary coded as '0001 0 1 01 100 0 0101'
			LOAD W5, [+W12] ; W12 = W12 + 2 then the 16-bits value at address defined by value of register W12 is stored in register W5. Binary coded as '0001 0 1 01 100 1 0101'
			STORE [-W12], B5 ; W12 = W12 - 1 then the 8-bits value of register 'B5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 1 11 100 0 0101'
			STORE [-W12], W5 ; W12 = W12 - 2 then the 16-bits value of register 'W5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 1 11 100 1 0101'
			LOAD B5, [-W12] ; W12 = W12 - 1 then the 8-bits value at address defined by value of register W12 is stored in register B5. Binary coded as '0001 0 1 11 100 0 0101'
			LOAD W5, [-W12] ; W12 = W12 - 2 then the 16-bits value at address defined by value of register W12 is stored in register W5. Binary coded as '0001 0 1 11 100 1 0101'
			STORE [W12+], B5 ; do W12 = W12 + 1 after the 8-bits value of register 'B5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 0 01 100 0 0101'
			STORE [W12+], W5 ; do W12 = W12 + 2 after the 16-bits value of register 'W5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 0 01 100 1 0101'
			LOAD B5, [W12+] ; do W12 = W12 + 1 after the 8-bits value at address defined by value of register W12 is stored in register B5. Binary coded as '0001 0 0 01 100 0 0101'
			LOAD W5, [W12+] ; do W12 = W12 + 2 after the 16-bits value at address defined by value of register W12 is stored in register W5. Binary coded as '0001 0 0 01 100 1 0101'
			STORE [W12-], B5 ; do W12 = W12 - 1 after the 8-bits value of register 'B5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 0 11 100 0 0101'
			STORE [W12-], W5 ; do W12 = W12 - 2 after the 16-bits value of register 'W5' is stored in memory at address defined by value of register W12. Binary coded as '0001 1 0 11 100 1 0101'
			LOAD B5, [W12-] ; do W12 = W12 - 1 after the 8-bits value at address defined by value of register W12 is stored in register B5. Binary coded as '0001 0 0 11 100 0 0101'
			LOAD W5, [W12-] ; do W12 = W12 - 2 after the 16-bits value at address defined by value of register W12 is stored in register W5. Binary coded as '0001 0 0 11 100 1 0101'
	0010 d zzz yyy r xxxx : LOAD and STORE with addressing mode 1
		d: direction, 0 from memory to register (LOAD), 1 from register to memory (STORE)
		r: register size, value 0 for 8-bits register from B0 to B15, value 1 for 16-bits register from W0 to W15
		xxxx : register, if d=0  register is destination.If d=1 register is source, register B0 to B15 if r=0, register W0 to W15 if r=1
		yyy : register containing the address. the register is yyy+8 to designate W8 to W15.
		zzz : register containing the offset address. the register is zzz to designate W0 to W7
		Assembler template :
			STORE [Wy + Wz], Bx ; The 8-bits value of register Bx is stored in memory at address resulting of the sum of Wy and  Wz. x from 0 to 15, y from 8 to 15, z from 0 to 7
			STORE [Wy + Wz], Wx ; The 16-bits value of register Bx is stored in memory at address resulting of the sum of Wy and  Wz. x from 0 to 15, y from 8 to 15, z from 0 to 7
			LOAD Bx, [Wy + Wz] ; store the 8-bits value stored in memory at address defined by the sum of value of register Wy and Wz to register Bx. x from 0 to 15,  y from 8 to 15, z from 0 to 7
			LOAD Wx, [Wy + Wz] ; store the 16-bits value stored in memory at address defined by the sum of value of register Wy and Wz to register Wx. x from 0 to 15,  y from 8 to 15, z from 0 to 7
		Assembler examples :
			STORE [W12 + W6], B5 ; The 8-bits value of register B5 is stored in memory at address resulting of the sum of W12 and  W6. Binary coded as '0010 1 110 100 0 0101'
			STORE [W12 + W6], W5 ; The 16-bits value of register W5 is stored in memory at address resulting of the sum of W12 and  W6. Binary codded as '0010 1 110 100 1 0101'
			LOAD B5, [W12 + W6] ; store the 8-bits value stored in memory at address defined by the sum of value of register W12 and W6 to register B5. Binary coded '0010 0 110 100 0 0101'
			LOAD W5, [W12 + W6] ; store the 16-bits value stored in memory at address defined by the sum of value of register W12 and W6 to register W5. Binary coded '0010 0 110 100 1 0101'
	0011 d iii yyy r xxxx: LOAD and STORE with addressing mode 2
		d: direction, 0 from memory to register (LOAD), 1 from register to memory (STORE)
		r: register size, value 0 for 8-bits register from B0 to B15, value 1 for 16-bits register from W0 to W15
		xxxx : register, if d=0  register is destination.If d=1 register is source, register B0 to B15 if r=0, register W0 to W15 if r=1
		yyy : register containing the address. the register is yyy+8 to designate W8 to W15.
		iii : value between 0 to 7, used as immediate with value computed as (iii + 1) << t. if r = 0 then immediate  is between 1 and 8. if r = 1 then immediate is 2 or 4 or 5 or 6 or 8 or 10 or 12 or 14 or 16. The immediate value is designated as 'imm'
		Assembler template :
			STORE [Wy + imm], Bx ; The 8-bits value of register Bx is stored in memory at address resulting of the sum of Wy and imm (from 1 to 8). x from 0 to 15, y from 8 to 15.
			STORE [Wy + imm], Wx ; The 16-bits value of register Wx is stored in memory at address resulting of the sum of Wy and imm (value 2 or 4 or 6 or 8 or 10 or 12 or 14 or 16). x from 0 to 15, y from 8 to 15.
			LOAD Bx, [Wy + imm] ; Store the 8-bits value in memory at address defined by the sum of Wy and imm (from 1 to 8) to register Bx. x from 0 to 15, y from 8 to 15.
			LOAD Wx, [Wy + imm] ; Store the 16-bits value in memory at address defined by the sum of Wy and imm (value 2 or 4 or 6 or 8 or 10 or 12 or 14 or 16) to register Wx. x from 0 to 15, y from 8 to 15.
		Assembler examples :
			STORE [W12 + 3], B10 ; The 8-bits value of register B10 is stored in memory at address resulting of the sum of W12 and 3. Binary coded as '0011 1 010 100 0 1010'.
			STORE [W12 + 4], W10 ; The 16-bits value of register W10 is stored in memory at address resulting of the sum of W12 and 4. Binary coded as '0011 1 001 100 1 1010'
			LOAD B10, [W12 + 3] ; Store the 8-bits value in memory at address defined by the sum of W12 and 3 to register B10. Binary coded as '0011 0 010 100 0 1010'
			LOAD W10, [W12 + 4] ; Store the 16-bits value in memory at address defined by the sum of W12 and 4 to register W10. Binary coded as '0011 0 001 100 1 1010'
	1 d iii zzz yyy r xxxx: LOAD and STORE with addressing mode 3
		d: direction, 0 from memory to register (LOAD), 1 from register to memory (STORE)
		r: register size, value 0 for 8-bits register from B0 to B15, value 1 for 16-bits register from W0 to W15
		xxxx : register, if d=0  register is destination.If d=1 register is source, register B0 to B15 if r=0, register W0 to W15 if r=1
		yyy : register containing the address. the register is yyy+8 to designate W8 to W15.
		zzz : register containing the offset address. the register is zzz to designate W0 to W7
		iii : value between 0 to 7, used as immediate with value computed as (iii + 1) << t. if r = 0 then immediate  is between 1 and 8. if r = 1 then immediate is 2 or 4 or 5 or 6 or 8 or 10 or 12 or 14 or 16. The immediate value is designated as 'imm'
		Assembler template :
			STORE [Wy + Wz + imm], Bx ;  The 8-bits value of register Bx is stored in memory at address resulting of the sum of Wy and Wz and imm (from 1 to 8). x from 0 to 15, y from 8 to 15. z from 0 to 7
			STORE [Wy + Wz + imm], Wx ;  The 16-bits value of register Wx is stored in memory at address resulting of the sum of Wy and Wz and imm (value 2 or 4 or 6 or 8 or 10 or 12 or 14 or 16). x from 0 to 15, y from 8 to 15. z from 0 to 7
			LOAD Bx, [Wy + Wz + imm] ; Store the 8-bits value in memory at address defined by the sum of Wy and Wz and imm (from 1 to 8) to register Bx. x from 0 to 15, y from 8 to 15, z from 0 to 7.
			LOAD Wx, [Wy + Wz + imm] ; Store the 16-bits value in memory at address defined by the sum of Wy and Wz and imm (value 2 or 4 or 6 or 8 or 10 or 12 or 14 or 16) to register Wx. x from 0 to 15, y from 8 to 15, z from 0 to 7.
		Assembler examples :
			STORE [W12 + W6 + 3], B10 ;  The 8-bits value of register B10 is stored in memory at address resulting of the sum of W12 and W6 and 3. Binary coded as  '1 1 010 110 100 0 1010'
			STORE [W12 + W6 + 4], W10 ;  The 16-bits value of register W10 is stored in memory at address resulting of the sum of W12 and W6 and 4. Binary coded as '1 1 001 110 100 1 1010'
			LOAD B10, [W12 + W6 + 3] ; Store the 8-bits value in memory at address defined by the sum of W12 and W6 and 3 to register B10. Binary coded as '1 0 010 110 100 0 1010'
			LOAD W10, [W12 + W6 + 4] ; Store the 16-bits value in memory at address defined by the sum of W12 and W6 and 4 to register W10. Binary coded as '1 0 001 110 100 1 1010'

3.3) Immediate assignment
	0100 iiii iiii xxxx: 8-bits immediate value to a register from B0 to B15
		iiii iiii: 8 bits immediate value (designated as 'imm')
		xxxx: Register Bx, x from 0 to 15
		Assembler template :
			SET Bx, imm ; set imm immediate value to register Bx
		Assembler examples :
			SET B10, 42 ; set the value 42 to register B10
			SET B9, -1 ; set the value 255 (-1 in signed 8-bits value) to register B9
	0101 iiii iiii xxx h: 8-bits immediate value to LSB/MSB part of register W8 to W15
		iiii iiii: 8 bits immediate value (designated as 'imm')
		xxx: Register W(x+8), x from 0 to 7. Designate the a register from W8 to W15
		h: 0 LSB of W(x+8), 1 MSB of W(x+8)
		Assembler template :
			LSB Wx, imm ; set imm immediate as LSB value of register Wx, x from 8 to 15
			MSB Wx, imm ; set imm immediate as MSB value of register Wx, x from 8 to 15
		Assembler examples :
			LSB W10, 42 ; set the LSB value of register W10 to 42.
			MSB W10, -1 ; set the MSB value of register W10 to  255 (-1 in signed 8-bits value)
